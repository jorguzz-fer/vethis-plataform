# Vethis — Educação Médica Veterinária

Mockup online da **Vethis**, uma plataforma de educação continuada em medicina
veterinária. Este repositório contém uma página de demonstração (landing page)
que mostra como a plataforma fica online: identidade visual, catálogo de cursos,
o aplicativo móvel e o painel para clínicas e instituições.

## O que tem aqui

- **`index.html`** — página única, autossuficiente (HTML + CSS + JS inline, sem
  dependências externas). Basta abrir no navegador.
- **`assets/`** — versões otimizadas do logotipo (marca e wordmark, fundo
  transparente).
- **`Logo.png`** — logotipo original da marca.

## Seções da demonstração

1. **Hero** — proposta de valor com mockup do app.
2. **Especialidades** — trilhas por área clínica (cardiologia, cirurgia,
   dermatologia, anestesiologia, imagem, exóticos, odontologia, emergência).
3. **Cursos em destaque** — cartões de curso com instrutor, carga horária e preço.
4. **App móvel** — telas do aplicativo (player de aula e progresso).
5. **Para clínicas** — painel de gestão de aprendizagem (KPIs, gráfico de
   conclusões e turmas ativas).
6. **Corpo docente**, depoimento e chamada para ação.

## Identidade visual

- **Cores:** verde-floresta `#0E4A38` (marca), dourado `#B8935A` (acento),
  neutros com leve viés esverdeado, fundo claro `#F4F7F3`.
- **Tipografia:** serifada (Iowan Old Style / Palatino / Georgia) para títulos;
  sans do sistema para interface e texto corrido.
- **Temas:** claro e escuro, com alternância pelo botão no cabeçalho.

## Como visualizar

```bash
# abra diretamente
open index.html          # macOS
xdg-open index.html      # Linux

# ou sirva localmente
python3 -m http.server 8000
# depois acesse http://localhost:8000
```

> Conteúdo de demonstração (cursos, instrutores e números são fictícios).
