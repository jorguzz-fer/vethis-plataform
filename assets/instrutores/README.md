# Fotos do corpo docente

Coloque aqui as fotos dos instrutores com **exatamente estes nomes**. O site
carrega automaticamente; se um arquivo não existir, o card mostra a inicial do
nome como fallback (nenhum código precisa ser alterado).

| Arquivo               | Instrutor           | Especialidade    |
|-----------------------|---------------------|------------------|
| `ricardo-mendes.jpg`  | Dr. Ricardo Mendes  | Cardiologia      |
| `ana-faria.jpg`       | Dra. Ana B. Faria   | Cirurgia         |
| `carlos-nunes.jpg`    | Dr. Carlos Nunes    | Dermatologia     |
| `lucia-prado.jpg`     | Dra. Lúcia Prado    | Anestesiologia   |

## Recomendações
- Formato: `.jpg` (ou troque a extensão no `index.html`, seção "instrutores").
- Proporção **quadrada** (ex.: 400×400 px). A foto é exibida em um círculo com
  recorte central (`object-fit: cover`).
- Peso leve (< 150 KB por foto) para carregamento rápido.

Para trocar pelas fotos reais depois, basta **substituir os arquivos** mantendo
os mesmos nomes.
