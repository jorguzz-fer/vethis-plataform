# Fotos dos instrutores

A seção "Instrutores" do site (`components/site/instrutores.tsx`) usa estas
fotos. Enquanto o arquivo não existe, o card mostra a inicial do instrutor
(fallback automático) — nada quebra.

Coloque cada foto aqui com **exatamente** este nome (quadrada, de preferência
≥ 512×512; o recorte circular é feito por CSS):

| Arquivo               | Instrutor                         |
| --------------------- | --------------------------------- |
| `ricardo-mendes.webp` | Dr. Ricardo Mendes — Cardiologia  |
| `ana-faria.webp`      | Dra. Ana B. Faria — Cirurgia      |
| `carlos-nunes.webp`   | Dr. Carlos Nunes — Dermatologia   |
| `lucia-prado.webp`    | Dra. Lúcia Prado — Anestesiologia |

Pode subir pelo GitHub (**Add file → Upload files**) direto nesta pasta.
Depois, **rebuild + redeploy do site** no Coolify.
